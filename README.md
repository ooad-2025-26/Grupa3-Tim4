# GameHub Manager

GameHub Manager je web aplikacija razvijena u ASP.NET Core MVC tehnologiji za upravljanje gaming centrom. Sistem omogućava upravljanje korisnicima, uređajima, rezervacijama, sesijama igranja, prijavom kvarova, narudžbama i praćenje esport takmičenja kroz jednostavan i moderan interfejs.

## Funkcionalnosti

- Upravljanje korisnicima i ulogama
- Kreiranje i pregled rezervacija
- Evidencija gaming uređaja
- Prijava i praćenje kvarova
- Slanje narudžbi i poruka zaposlenicima
- Upravljanje gaming sesijama
- Pregled i organizacija takmičenja
- Administratorski panel za upravljanje sistemom
